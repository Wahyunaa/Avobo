# Avobo
Avobo is a game where the aim is to guess which one is not a bomb</p>


## Required
- `Python3`
## How to use
- `git clone https://github.com/wahyunaa/avobo`
- `cd Avobo/avobo`
- `python avobo.py`


> **_NOTE:_** Not all terminals support showing emotes regularly
